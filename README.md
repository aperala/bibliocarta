# README

A ‘book-map’ social app that lets users log in, find books by place/setting, and save books to their ‘passport’ to create a map of books they have read keyed to places. 

* Ruby 2.2.1

## System dependencies: 
  * gem 'devise'
  * gem 'isbn_validation', '~> 1.2.0'
  * gem "paperclip", "~> 4.3"
